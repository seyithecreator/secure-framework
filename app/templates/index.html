End-to-End Encryption

Files are encrypted with AES-256 (Fernet) before hitting disk. Decryption only happens in memory at download time — never stored in plaintext.

Brute-Force Protection

Automatic IP blocking after failed login attempts across FTPS, SFTP, and the web interface. Configurable thresholds and block durations.

Config Integrity Auditing

SHA-256 baseline snapshots of your config files. Run an audit anytime to detect unauthorized changes or configuration drift.

FTPS & SFTP Servers

Run TLS-secured FTPS (pyftpdlib) and SSH-based SFTP (paramiko) alongside the web vault. All auth shares a single hardened user database.

Rate Limiting & HSTS

Flask-Limiter on every endpoint plus Talisman enforcing HSTS, strict CSP, and X-Frame-Options DENY — hardened out of the box.

Argon2 Password Hashing

Passwords are hashed with Argon2id — the 2015 Password Hashing Competition winner — resistant to GPU and side-channel attacks.
